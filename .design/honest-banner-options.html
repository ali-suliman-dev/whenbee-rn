Color usage · the fix

60 · 30 · 10 on this screen
60% neutral — page bg + surface chips/inputs carry the form.

30% ink — labels, the typed task, the number values.

10% accent, split by role:
• Indigo = the one action you take → the CTA + the current selection state (chips/guess). Exactly one filled indigo destination.
• Amber = the honey/ripen payoff → the honest estimate. It's the bee's learned truth, so it earns the true accent.

Splitting action (indigo) from insight (amber) is what kills the "everything's indigo" mush and gives the screen a clear focal order: type → pick → see the honest number → act.

Invariant check: amber is reserved for honey/ripen/reward/optimism. The honest estimate is the calibration payoff — fits. If you'd rather keep amber strictly for reward screens, option B (neutral) or E (indigo) stay on the safe side.
