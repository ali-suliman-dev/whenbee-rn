Whenbee — Start-By Plan redesign · 4 directions

The reverse planner: pick a finish time + tasks, Whenbee says when to start using how long things actually take you. Faithful to the dark theme tokens (#8275F0 indigo hero, amber reserved for honey/triage, 4-based spacing, card 16 radii). Every direction fixes the same broken universals — they differ in structure.

✅ Inline add — no modal ✅ Drag to reorder — press-hold-drag (reanimated) ✅ Reset / Re-plan action present ✅ Loggable rows — tap ▶ runs the real timer ✅ Build disabled until ≥1 task ✅ Amber-never-red triage verdict

A Timeline Canvas

The schedule is the screen. Tasks are time-blocks on a vertical axis, height ∝ duration. Drag a block to reorder, tap to expand / Start. Most alive, most visual.

Grounded in: Morgen / ClickUp drag-blocks-onto-time-axis timeboxing pattern.
