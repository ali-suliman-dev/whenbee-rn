D3 · Toybox Tactile — pinned. In context, correct states, + the shine question.

Can the shine be done in React Native? Yes, trivially — no library, no boxShadow risk. The fill is a rounded View; the shine is an absolutely-positioned child View (height 3, borderRadius 2, backgroundColor rgba(255,255,255,0.38), top 3, left/right 6). Pure Views on the GPU — Fabric-safe, no RN 0.81 boxShadow bug, works identically on Android. The card's extruded edge = the same View-layering trick AppButton's coin edge already uses, so the whole D3 look is proven tech in this codebase. The bar below uses the app's REAL anatomy: indigo hatch = your guess, amber = the honest top-up minutes, black tick = live elapsed (running only).
