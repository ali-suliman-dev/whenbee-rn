v6 — where "Now" sits

F2 pinned, plus the two title-line variants you asked for. Everything else is unchanged: real overlay, scrim, grabber, HH : MM pan-wheel, Done in the bottom-right.

The one thing placement decides. The footer slot fires an action — tap it and the wheel jumps to now. The title-line slot can also hold a state — it can show you the row is currently on Now, before you touch anything. That matters because Now is a live anchor (startAt: null, re-derives every render) while a spun time is fixed. If nothing on screen distinguishes them, reopening the sheet tomorrow tells you nothing about which one you're in.

Label

LabelRead
Use nowUnambiguously an action. Reads well in the footer (F2) and as a link (G1). Slightly odd inside a chip, where a verb implies the chip is a button rather than a state.
Now / Now · 09:05Reads as a value, which is right for a chip that can be selected. The appended clock also answers "what would that actually be?" without a tap. Best for G2, weak as a footer button — a lone "Now" beside "Done" looks like a second confirm.
Start nowClear, but "Start" collides with the timer — on a screen that plans a day rather than launching one, it risks reading as a play button. I'd avoid it here.

So the label follows the placement: F2/G1 → "Use now", G2 → "Now · 09:05".
