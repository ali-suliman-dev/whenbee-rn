Whenbee · Paywall redesign v1

The old paywall pitched one feature (calendar) for a nine-feature Pro bundle, arriving the same way from 15 different gates — so it read as random and undersold. New structure: a hero that adapts to the gate the user tapped, a scannable value-stack that represents the whole bundle, an anchored plan picker, and an honest trust block. No fabricated numbers, no guilt.
